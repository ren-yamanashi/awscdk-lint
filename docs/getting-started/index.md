# Getting Started

`awscdk-lint` ships the same CDK rule set as two packages. Pick the one matching the linter your project uses:

- [**Oxlint setup**](./oxlint/)
- [**ESLint setup**](./eslint/)

Both packages expose the same rules under the `awscdk/` namespace. After setup, see the [Rules reference](../rules/) for what each rule does.
