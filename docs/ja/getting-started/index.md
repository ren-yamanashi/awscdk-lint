# Getting Started

`awscdk-lint` は同じ CDK ルールセットを 2 つのパッケージで提供しています。プロジェクトで使用している linter に合わせて選択してください。

- [**Oxlint のセットアップ**](./oxlint/)
- [**ESLint のセットアップ**](./eslint/)

どちらのパッケージも、`awscdk/` 名前空間で同じルールを公開しています。セットアップ後、各ルールの詳細は [ルールリファレンス](../rules/) を参照してください。
